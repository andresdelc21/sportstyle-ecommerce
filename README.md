# 🏆 SportStyle Ecommerce

Sistema de comercio electrónico desarrollado en PHP, MySQL, HTML, CSS y JavaScript para la venta de indumentaria y accesorios deportivos.

## 📌 Descripción

SportStyle es un proyecto personal orientado al desarrollo de una tienda online moderna, incorporando funcionalidades habituales de los e-commerce profesionales.

El objetivo del proyecto es aplicar conocimientos de desarrollo web, bases de datos, experiencia de usuario y gestión de productos mediante una aplicación completa construida desde cero.

## 🔧 Instalación

1. Clonar el repositorio:

```bash
git clone https://github.com/progrmair21/sportstyle-ecommerce.git
```

2. Copiar el proyecto dentro de:

```text
xampp/htdocs/
```

3. Crear una base de datos llamada:

```text
sportstyle
```

4. Configurar la conexión en:

```php
config/conexion.php
```

5. Iniciar Apache y MySQL desde XAMPP.

6. Abrir en el navegador:

```text
http://localhost/sportstyle
```
